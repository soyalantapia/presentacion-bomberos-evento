## Navegación

- `← →` · `Space` · `Page Up/Down` — navegar entre slides
- `Home` / `End` — primer / último slide
- Swipe en mobile
- Click en los dots laterales para saltar directo

## Características

- 47 slides full-screen responsive (16:9)
- Zero dependencies — un único HTML con todo inline
- Animaciones sutiles clicker-friendly (no dependen de clicks)
- Logos oficiales de tools (ChatGPT, Claude, Meta AI, Canva, Gemini)
- Mockups de celular por cada herramienta
- Timeline histórica con thumbnails ilustrados
- 12 ilustraciones SVG originales
- Respeta `prefers-reduced-motion`

## Stack

- HTML + CSS + JS vanilla
- Poppins (Google Fonts)
- SVG inline — zero imágenes externas
